The OR Gate

Congratulations on making it to the third lesson. By this point you will start to notice that many of the logic gates are very similar with only a few small differences. Another basic logic gate like the AND gate is the OR gate. Like the AND gate, the OR gate also has two inputs, but the difference is that the OR gate only needs at least one input to be true to produce an output value of true.
