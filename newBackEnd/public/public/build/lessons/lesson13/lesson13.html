The NOR Gate

Now we will move from the OR gate to the NOR gate. Assuming you have a good understanding of the OR gate, the NOR gate should be easy to learn. The output of a NOR gate will produce the opposite output that an OR gate would produce. The NOR gate will only output true if both inputs are false. If one or both inputs are true, the NOR gate will output false. Look over the NOR gate symbol and truth table then begin the challenge when you are ready.
